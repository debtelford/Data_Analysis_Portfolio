# SQL projects

This folder contains SQL projects and queries used for data analysis practice. Currently, it features queries on the **Northwind** database, focused on sales analysis.

## Project 1: northwind 
Source data: Northwind for SQLite3, downloaded from https://github.com/jpwhite3/northwind-SQLite3

The `northwind_sales_queries.sql` file includes a series of SQL queries exploring sales data, including:

- Monthly sales trends (all years)
- Top 10 customers by revenue 2020
- Top 20 products by revenue 2020
- Average Time to Ship by Carrier 2020
- Employee performance 2020

These queries demonstrate use of:

- joins
- aggregations and grouping
- filtering and date functions
- ordering and limiting results
- basic calculations and formatting
